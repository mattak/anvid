# Anvid

Android Video Server

# install

    bundle install --path vendor/bundle

# run

    bundle exec rackup

Then you can access http://localhost:1028 .

# capture & share

    $ curl http://localhost:1028/sh/anvid > /your/bin/anvid
    $ chmod a+x /your/bin/anvid

record video.

    $ anvid record

capture screen.

    $ anvid capture

# Rodeo

Android Video Server

# install

    bundle install --path vendor/bundle

# run

    bundle exec rackup

Then you can access http://localhost:1028 .

# capture & share

    $ curl http://localhost:1028/sh/rodeo > /your/bin/rodeo
    $ chmod a+x /your/bin/rodeo

record video.

    $ rodeo record

capture screen.

    $ rodeo capture
